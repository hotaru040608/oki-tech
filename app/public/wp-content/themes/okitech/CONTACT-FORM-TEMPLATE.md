# Contact Form 7 お問い合わせフォームテンプレート

このテンプレートを使用して、OkiTechテーマ用のお問い合わせフォームを作成できます。

## セットアップ手順

1. WordPress管理画面にログイン
2. **お問い合わせ** → **コンタクトフォーム** → **新規追加**
3. 以下のフォームコードをコピー＆ペースト
4. フォーム名を「お問い合わせフォーム」などに設定
5. **保存**をクリック

## フォームコード

```
<label>お名前 <span class="required">*</span>
    [text* your-name]</label>

<label>メールアドレス <span class="required">*</span>
    [email* your-email]</label>

<label>電話番号（任意）
    [tel your-tel]</label>

<label>お問い合わせ種別
    [select your-subject include_blank "イベントについて" "コミュニティについて" "その他"]</label>

<label>お問い合わせ内容 <span class="required">*</span>
    [textarea* your-message]</label>

[submit "送信する"]
```

## メール設定

### メール（1）: 管理者への通知

**宛先**: `[your-email]` または管理者のメールアドレス

**件名**: `お問い合わせがありました: [your-subject]`

**メッセージ本文**:
```
以下の内容でお問い合わせがありました。

お名前: [your-name]
メールアドレス: [your-email]
電話番号: [your-tel]
お問い合わせ種別: [your-subject]

お問い合わせ内容:
[your-message]

---
このメールは、OkiTechのお問い合わせフォームから送信されました。
```

### メール（2）: 自動返信メール

**宛先**: `[your-email]`

**件名**: `お問い合わせありがとうございます - OkiTech`

**メッセージ本文**:
```
[your-name] 様

この度は、OkiTechにお問い合わせいただき、誠にありがとうございます。

以下の内容でお問い合わせを受け付けました。

【お問い合わせ内容】
お問い合わせ種別: [your-subject]

お問い合わせ内容:
[your-message]

通常、2営業日以内にご返信いたします。
しばらくお待ちください。

ご不明な点がございましたら、このメールに返信していただくか、
再度お問い合わせフォームよりご連絡ください。

---
OkiTech
沖縄のAI・SNS・プログラミングを学べる場所
```

## 追加設定

### メッセージのカスタマイズ

フォーム設定の「メッセージ」タブで、以下のメッセージをカスタマイズできます：

- **送信成功時のメッセージ**: 「お問い合わせありがとうございます。メッセージは正常に送信されました。」
- **送信失敗時のメッセージ**: 「送信に失敗しました。もう一度お試しください。」
- **バリデーションエラーメッセージ**: 各フィールドのエラーメッセージ

### スパム対策

1. **reCAPTCHA v3** を設定（推奨）
   - **お問い合わせ** → **統合** → **reCAPTCHA v3**
   - Google reCAPTCHAのサイトキーとシークレットキーを設定

2. **Akismet** を使用
   - Akismetプラグインをインストール・有効化

## テーマとの統合

フォームを作成後、以下の手順でテーマに統合します：

1. **外観** → **カスタマイズ** → **お問い合わせフォーム**
2. 作成したフォームを選択
3. **公開**をクリック

これで、お問い合わせページ（`/contact/`）にフォームが表示されます。

## トラブルシューティング

### フォームが表示されない場合

1. Contact Form 7プラグインが有効化されているか確認
2. フォームが公開状態になっているか確認
3. テーマのカスタマイザーでフォームIDが正しく設定されているか確認

### メールが届かない場合

1. WordPressのメール送信設定を確認
2. サーバーのメール送信機能が有効か確認
3. スパムフォルダを確認
4. SMTPプラグイン（例: WP Mail SMTP）の使用を検討
